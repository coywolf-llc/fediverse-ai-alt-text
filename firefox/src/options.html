AI Alt Text for the Fediverse

Generate image alt text for Mastodon with your own Claude API key. Nothing is sent anywhere except api.anthropic.com.

Mastodon instances

Works on any Mastodon instance — add the domain of each one you use, including self-hosted ones (e.g. mastodon.social, henshaw.social, coywolf.social). Your browser will ask permission to run on that site; the button only appears there. After adding one, reload its tab once.

    Anthropic API key

    Your key is stored only on this device (browser.storage.local) and is sent only to api.anthropic.com. Get a key at console.anthropic.com.

    Validating costs a few input tokens — well under a thousandth of a cent.
